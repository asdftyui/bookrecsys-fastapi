# 책 추천 웹 사이트

## 소개
책 추천 대회에 사용한 Catboost model을 기반으로 사용자의 입력을 받아 책을 추천하는 웹 사이트 구현

## 프로젝트 결과

## 구동 방법
### 프론트엔드
```shell
streamlit run frontend.py
```

### 백엔드
```shell
poetry shell
python backend.py
```

## 사용 기술
프론트엔드: streamlit
<br>
백엔드: fastapi
